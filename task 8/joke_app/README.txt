Random Joke App - Flask
=======================

HOW TO RUN:
-----------
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://localhost:5000

FEATURES:
---------
- Random jokes from free API (no API key needed)
- Filter by category: All, Programming, General, Knock-Knock
- Reveal punchline button
- Joke counter
- Dynamic frontend with Flask backend

API USED:
---------
https://official-joke-api.appspot.com  (free, no key required)
