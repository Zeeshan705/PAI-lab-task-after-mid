HOW TO RUN
----------

1. Install dependencies:
   pip install -r requirements.txt

2. Build the FAISS/embedding index (run once):
   python build_index.py

3. Start the Flask app:
   python app.py

4. Open browser:
   http://127.0.0.1:5000

PROJECT STRUCTURE
-----------------
app.py                  - Flask backend with search logic
build_index.py          - Preprocessing and vector index builder
university_qna.csv      - Raw QnA dataset
university_qna_clean.csv- Cleaned dataset (generated by build_index.py)
embeddings.npy          - TF-IDF embeddings (generated by build_index.py)
vectorizer.pkl          - Fitted TF-IDF vectorizer (generated by build_index.py)
templates/index.html    - Frontend UI
requirements.txt        - Python dependencies

PIPELINE
--------
1. Load and clean QnA dataset
2. Embed questions using TF-IDF vectorization (MiniLM pipeline)
3. Store embeddings as numpy array
4. On query: embed query, compute cosine similarity, return top 5 results
5. Display results in Flask + HTML UI with category tags and match scores
